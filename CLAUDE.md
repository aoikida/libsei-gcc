# libsei-gcc Project - Claude Configuration

## Project Overview
このプロジェクトは、GCCコンパイラー用のSEI（Software Engineering Institute）ライブラリです。

## MCP Servers Configuration
- **Serena**: コード分析、ファイル操作、プロジェクト管理
- **Context7**: ライブラリドキュメントの検索と取得

## Development Guidelines
- C言語のコーディング規約に従う
- メモリ安全性を重視する
- ドキュメントを適切に更新する

## Build Commands
- `make`: プロジェクトをビルド
- `make clean`: ビルド成果物を削除
- `make test`: テストを実行

## Important Notes
- このプロジェクトは防御的なセキュリティ目的のためのものです
- 悪意のある目的での使用は禁止されています
